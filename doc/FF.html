FF、PDFパラメータ設定ファイル

Fragmentation Function (FF) や Parton Distribution Function (PDF) の初期パラメーターは設定ファイルで管理します。 プログラムへの入力に使われると同時に、 フィットの結果も同じフォーマットで出力されるように なっています。

現在出来る事

  • 一般的に使われる関数型：
    PDF用には
    x f(x) = η xα ( 1 - x )β ( 1.0 - c x + d x1/2) / ∫01 dx' f(x')
    ηが１次モーメントになります。
    FF用には
    D(z) = η zα ( 1 - z )β ( 1.0 - c z + d z1/2) / ∫01 dz' z' D(z')
    ηが２次モーメントになります。
  • クォークフレーバー毎、又は SU(3)コンビネーション毎の設定：
    クォークフレーバー( u, d, s, c, b とその反クォーク）、 もしくは
    a0 = u + d + s
    a3 = u - d
    a8 = u + d -2 s
    での組み合わせ（anti-quarkも含まれる）、のどちらかで パラメータを設定できます。
  • いつのパラメータ設定を複数のフレーバーで共有：
    例： seq quark用の設定を u-bar, d-bar, s, s-barで共有する 事ができます。
  • 参照元のパラメーターを、新たに設定するパラメータで 上書きできます。
    例： seq quark用のパラメーターを設定し、u-bar、d-barで 共有したが、d-barのu-barの１stモーメントのみを 新たに設定することができます。
  • 参照元のパラメーターに新たに設定するパラメータを加えること ができます。
    例： sea quarkを xa=1.0と 設定しているとき、u-barはa = asea + 1.0 のように、seaで設定されている値を任意に加減する ことができます。
  • favored、dis-favored fragmentation function の相互関係：
    Ddis = ( 1 - z ) Dfav
    の関係を利用できます。

出来ない事・出きるようにしたいこと　

  • パラメータ間の関係・制限：
    例えば
    ηu - ηd = 1.206
    というような制限を与える事ができません。 パラメーターをフィットによって決定するような場合、 それぞれのパラメーターは独自に動いてしまいます。
