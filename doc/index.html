ライブラリについて

基本ライブラリ

Utility
各種のツールが含まれています。 Argumentsクラス、Configクラス、 Timerクラス等。
Tranform
特殊関数および積分ルーチン等が含まれています。 データキャッシュ用のクラスとして LookupTableテンプレートクラス（ディスクベース）、 DataCacheクラス（メモリベース）があります。
QCD
QCDの基本量を取り扱うためのクラスが含まれています。

パートン分布関数・フラグメンテーション関数

Evolution
PDF等の発展方程式計算のための基本クラスが含められています。
unpolPDF
Unpolarized PDFのSplitting Function の実装
polPDF
Polarized PDFのSplitting Function の実装
Fragmentation
Unpolarized Fragmentation Function (FF) Splitting Function の実装
polFrag
Polarized FF Splitting Function の実装

断面積・構造関数計算

Xsec
断面積・構造関数を計算するための基本クラスが含められています。
FragXsec_T
電子・陽電子消滅でのハドロン生成断面積 σπ, K の実装
unpolDIS
非偏極深非弾性散乱実験での構造関数 F1(x,Q2)の実装
polDIS
偏極深非弾性散乱実験での構造関数 g1(x,Q2)の実装
unpolSIDIS
非偏極深非弾性散乱実験(Semi-inclusive 測定)での 構造関数 F1h(x,Q2)の実装
polSIDIS
非偏極深非弾性散乱実験(Semi-inclusive 測定)での 構造関数 g1h(x,Q2)の実装
IntegXsec
断面積・構造関数等の計算をx空間で 行うためのライブラリ
IntegUnpolDIS
F1(x,Q2)の計算を x空間で 行うためのライブラリ
IntegPolDIS
g1(x,Q2)の計算を x空間で 行うためのライブラリ

その他

StrFncParam
構造関数>F2(x,Q2)や、 光子吸収断面積 σγ* N の Parametrization の実装
ExpData
実験データーを取り扱うためのクラスライブラリ
PDFLIB
各種 Parton Distribution Function Parameterization の実装
Fitting
実験データ(ExpData)を計算モデル(Xsec)を利用してフィットする ためのライブラリ。 ROOTのTMinuit を利用。
browser
ROOTをつかって、 分布関数・構造関数・実験データ等を表示するツール。
example
各種ライブラリのテスト・使用例等
