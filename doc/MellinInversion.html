逆Mellin変換

逆Mellin変換は
g( x ) = 1/2πi ∫C x-z f( z ) dz
このライブラリでとりあつかうPDF、FF、断面積、構造関数等は 実軸対称なので、積分経路を上半面のみで考える事で、
g( x ) = 1/π ∫C' Im( x-z f( z ) dz )
とする事ができます。 積分経路は任意であるので、数値計算の精度・計算速度の両面から 最適化を試みます。
直線積分経路
積分経路として、 ( c0, 0.0 )を開始点にとり、 右図にあるように、実軸からの角度φの方向に 長さlの直線をとります。

開始点は逆Mellin変換する関数のもつ特異点よりも右側にとります。 右側に移動すればするほど、より長い積分経路が必要です。

実際のプログラムでは、 開始点c0と、角度φ、 経路長lを設定します。 InverseMellin::setParameterメソッドで c0と、角度φ、 InverseMellin::upperメソッドで経路長を設定します。 それぞれのメソッドの引数については、 クラスリファレンスマニュアルを参照してください。

最降下曲線積分経路
z(t) = c0 + c1 t2 + i t
最降下曲線( Steepest Descent )に沿って積分を行う事で、 より効率的に積分値を得る事ができます。 この場合は、被積分関数をさらに式変形し、 ラゲール積分を行います。
