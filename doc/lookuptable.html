Lookup Tableについて

計算結果をキャッシュしておくためのLookup Tableが実装されています。 template < class Key, class Data > LookupTable< Key, Data > テンプレートクラスを利用します。コンストラクタは 
template < class Key, class Data > 
LookupTable< Key, Data >( Key& key, Data& data, const string& name );
です。データベースファイルは ${HOME}/.mellin/cache/nameとして作られます。 詳しい利用方法はリファレンスマニュアル、CPolyGamma.{hh,cc}等の 特殊関数のソースを参照してください。
